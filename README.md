# julia-cli-class-hello-world

## Description
A POC for julia scripting.

## Demonstrates
- class/struct
  - constructor
  - property
- variables in `println` 

## Tech stack
- julia

## Docker stack
- julia:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
